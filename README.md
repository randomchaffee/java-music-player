# java music player
A simple Java music player with albums, songs, and playlists. Uses file I/O and object serialization for persistence.

Created November 11, 2025
--- 
## Features

- **Album & Song Management**
  - Reads album metadata and song lists from `.txt` files.
  - Displays all songs and albums with clean formatting.

- **Playlist System**
  - Create and manage playlists.
  - Add or remove songs.
  - Saves playlists using Java object serialization.

- **Data Persistence**
  - Automatically saves and loads albums, songs, and playlists from disk.

- **File I/O + OOP**
  - Demonstrates practical use of classes, objects, encapsulation, and file handling.
